
INSTRUCCIONES:

1. Coloca el archivo `sigma.min.js` dentro de esta carpeta.
   Descárgalo desde aquí: https://unpkg.com/sigma@2/build/sigma.min.js
   (Click derecho > Guardar como...)

2. Abre el archivo `index.html` en tu navegador (doble clic).

3. Si no carga el grafo, abre un servidor local:
   - Si tienes Python 3 instalado:
     > python -m http.server
   - Luego visita http://localhost:8000

4. (Opcional) Sube esta carpeta a GitHub Pages para tenerlo en línea.
